<h1>Netflix Movies & TV Shows Visualization</h1>

<p>
This project focuses on <strong>visualizing Netflix Movies and TV Shows data</strong> 
using Python and Matplotlib. It analyzes trends, genres, ratings, and release 
years to gain meaningful insights from the Netflix dataset.
</p>

<h2>Features</h2>
<ul>
    <li>Visualization of Movies vs TV Shows distribution</li>
    <li>Release year trend analysis</li>
    <li>Top genres analysis</li>
    <li>Ratings distribution visualization</li>
    <li>Clean and informative plots using Matplotlib</li>
</ul>

<h2>How the Project Works</h2>
<ul>
    <li>Load and preprocess the Netflix dataset</li>
    <li>Handle missing values and format data</li>
    <li>Analyze key attributes like type, genre, rating, and year</li>
    <li>Create visualizations using Matplotlib</li>
</ul>

<h2>Purpose</h2>
<p>
This project was created to practice data analysis and data visualization 
techniques using Python. It helps in understanding real-world dataset 
handling and storytelling through visual insights.
</p>

<h2>Technologies Used</h2>
<ul>
    <li>Python</li>
    <li>Matplotlib</li>
    <li>Pandas</li>
</ul>
